Full Stack Development Nozama Boutique

Collaborated with team members to produce a web app for a hypothetical clothing purveyor, including payment authorization utilizing Stripe. Users can view individual product info, add products to a cart, and then submit their order for processing. Built a back end server with Express.js that utilizes MongoDB as its database, while front end functionality was created using Bootstrap, LESS, Handlebars and JavaScript.

A web application that allows users to create and save unique cheese plates. I built a custom back end API that uses a Ruby on Rails framework as well as a Bootstrap front end. Users can create one or more plates, select cheeses from a library of cheeses, and save their plates for future reference.
